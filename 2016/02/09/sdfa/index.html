Hexo搭建个人博客

  • 其他

环境配置

  • 1 安装Node
  • 2 安装Git
  • 3 申请GitHub账号（需要激活邮箱，否则后面部署出现404页面）
  • 4 安装hexo 命令（Windows下）：$ npm install -g hexo

  • 5 初始化 创建一个文件夹，cd到该文件夹执行hexo init命令自动生成如下文件:

    本地生成静态页面（预览作用）

  • 在init该文件目录下执行 hexo g 生成静态页面

  • 继续执行 hexo server 命令启动本地服务
  • 浏览器输入 http://localhost:4000 可以看到所生成的个人博客（如果失败，环境配置不正确）

建立Repository关联GitHub上传页面

  • 在GitHub里创建仓库，仓库名必须是 用户名.github.io
  • 打开上图中的_config.yml文件(用Subilme编辑器)建立关联
  • 在最后添加
    deploy:
    type: git
    repository: git@github.com:zhongyuanwu/zhongyuanwu.github.io.git（刚刚创建仓库的ssh）
    branch: master
  • 执行 hexo g
  • 执行 hexo d 在GitHub上发现如下

    在网页上打开 “你的GitHub用户名.github.io”链接出现

    表示成功，如果出现404，请检查GitHub邮箱是否验证

配置主题

  • 我所用的主题Yilia
    点入链接下载，把该文件复制到Themes目录下
    打开首页的_config.yml文件中 拉到最后 修改theme为 yilia，执行命令hexo g hexo d 或者在本地生成静态页面，
  • 在主题文件里的_config.yml中可以配置 头像（URL可以是本地的，图片文件放在首页的source目录下，可以是一个外部图片连接） 在首页的_config.yml文件中可以配置姓名 标题
  • 在首页的source目录下_posts中可以看到 .mc文件用subilme打开发现与博客界面一致
  • 把写好的Markdown文件放入source中的_posts中，部署即可
